# rmst-truncation-review
This repository store the extracted data (list of eligible RCTs) from a systematic literature review investigating the selection the truncation time for Restricted Mean Survival Time analysis.

- review_01012026.csv: list of included RCTs
- PRISMA_flowchart_sys_review_RMST.pdf: List of search database with the searching keywords and flowchart of inclusion/exclusion criteria.
